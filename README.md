# Lista de la Compra Compartida

Aplicación de lista de la compra colaborativa desarrollada en Android utilizando **Kotlin, Jetpack Compose y Firebase**.  
Permite gestionar listas personales y compartidas, organizadas mediante categorías y productos, con sincronización en tiempo real.

---

## Características principales

### Listas personales y compartidas

- Cada usuario dispone de sus propias listas.
- Una lista puede compartirse para que otros usuarios puedan editarla en tiempo real.
- Las listas compartidas aparecen agrupadas igual que en Gmail/Drive (listas propias y listas compartidas con el usuario).

---

### Categorías

- Cada lista incluye categorías personalizables.
- Las categorías pueden crearse, editarse y eliminarse.
- Ejemplos: *Lácteos*, *Frutas*, *Limpieza*, *Carnes*, etc.

---

### Productos

- Creación, edición y eliminación de productos dentro de cada categoría.
- Marcar como completado sin necesidad de borrar el producto.
- Contador visual de productos pendientes.

---

## Funcionalidad extra implementada

- Cada producto puede tener información asociada (como la tienda a la que pertenece, etiquetas, precio, etc).
